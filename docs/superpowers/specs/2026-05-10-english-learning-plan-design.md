# English Learning Plan: 6-Month Sentence + Grammar Based

## Goal

Internet industry professional (CET4/6, 13yr XP, can read docs, struggles with speaking grammar) aims to communicate fluently with foreign colleagues and prepare for remote job interviews within 6 months.

## Approach

- **Vocabulary + grammar** through sentences only — no standalone word lists
- Each vocab entry: 3 common meanings with usage, example sentences in internet/work contexts
- Each grammar point: example sentence → syntax breakdown → memorization tips
- Weekly rhythm: grammar points + vocab + sentence-making exercises + review
- Speaking practice handled independently by the user

## 6-Month Curriculum

| Month | Grammar | Vocabulary Domain |
|-------|---------|-------------------|
| 1 | Basic sentence structure, simple tenses (present/past/future), continuous tenses | Daily development terms (verbs/nouns) |
| 2 | Perfect tenses, perfect continuous, passive voice, modal verbs | Collaboration & communication |
| 3 | Noun clauses, relative clauses, adverbial clauses | System design & architecture |
| 4 | Conditionals, subjunctive mood, non-finite verbs (gerunds/infinitives/participles) | Product & data |
| 5 | Inversion, emphasis, ellipsis, articles & prepositions deep-dive | Business & interview |
| 6 | Comprehensive review, long-sentence breakdown, common error patterns | Remote work scenario integration |

## Weekly File Format

Each week-XX.md contains:

1. **Grammar** (1-2 points)
   - Example sentence
   - Syntax breakdown (subject, predicate, object, modifiers — word-by-word)
   - Understanding/memorization technique
   - 1-2 practice sentences with answer key

2. **Vocabulary** (15-20 words)
   - Each word with 3 common meanings
   - Example sentence per meaning (work/internet context preferred)
   - Usage notes where relevant

3. **Sentence-making exercises** — create sentences using this week's vocab + grammar

4. **Weekend review** — error review, commonly confused words

## Non-goals

- Speaking practice (user handles independently)
- Spaced-repetition algorithms or Anki decks
- Software/app development
- Progress tracking automation (manual only)
