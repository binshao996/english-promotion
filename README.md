## Weekly Learning Rhythm

The plan follows a steady, structured cadence — 4 weeks per month, 6 months total.

### Each Week

| Day | Focus | Time |
|-----|-------|------|
| **Mon–Tue** | Grammar — read the basic point, then deep-dive into 4–6 complex sentence breakdowns | ~45 min/day |
| **Wed–Thu** | Vocabulary — 50 words/day (each with 3 meanings + example sentences), 300 words total per week | ~30 min/day |
| **Fri** | Sentence-Making Practice — translate 10 Chinese prompts into English, compare with reference answers | ~30 min |
| **Sat** | Weekend Review — grammar fill-in, vocabulary self-test, structural analysis challenge | ~45 min |
| **Sun** | Rest + optional shadowing/reading | — |

### Each Month

```
Week 01 ─── New Grammar Topic A ─── 300 Words (Group 1–6)
Week 02 ─── New Grammar Topic B ─── 300 Words (Group 1–6)
Week 03 ─── New Grammar Topic C ─── 300 Words (Group 1–6)
Week 04 ─── Grammar Integration + Monthly Review ─── 300 Words (Group 1–6)
              │                              │
              └── All 3 grammar topics       └── All 1,200 monthly words
                  woven into complex sentences     available for review
```

### Weekly File Structure

Every `week-XX.md` follows the same layout:

```
week-XX.md
├── 1. Grammar
│   ├── Basic point (brief overview)
│   └── 4–6 complex sentence breakdowns (the core) — word-by-word syntax analysis
├── 2. Vocabulary (300 words, 6 groups × 50)
│   └── Each word: 3 meanings + example sentences
├── 3. Sentence-Making Practice (10 Chinese→English prompts)
└── 4. Weekend Review (grammar test, vocabulary test, structural challenge)
```

## 6-Month Roadmap

| Month | Phase | Grammar Focus | Total Words |
|-------|-------|--------------|-------------|
| 1 | **Foundation** | Sentence patterns, simple/continuous tenses | 1,200 |
| 2 | **Building** | Perfect tenses, passive voice, modal verbs | 1,200 |
| 3 | **Expanding** | Noun clauses, relative clauses, adverbial clauses | 1,200 |
| 4 | **Deepening** | Conditionals, subjunctive mood, non-finite verbs | 1,200 |
| 5 | **Refining** | Inversion, emphasis, articles & prepositions | 1,200 |
| 6 | **Mastering** | Advanced integration, long-sentence dissection, error patterns | 1,200 |
| **Total** | | | **7,200** |

### Monthly Goals

**Month 1 — Foundation**
- Master the 5 basic English sentence patterns
- Correctly use simple present, past, future, and continuous tenses
- Vocabulary: core dev terms + daily high-frequency words

**Month 2 — Building**
- Understand and use perfect tenses, passive voice, modal verbs
- Vocabulary: collaboration & communication + daily conversation

**Month 3 — Expanding**
- Master noun clauses, relative clauses, adverbial clauses
- Vocabulary: system design & architecture + daily life

**Month 4 — Deepening**
- Understand conditionals, subjunctive mood, non-finite verbs
- Vocabulary: product, data, social topics, international relations, philosophy, entrepreneurship

**Month 5 — Refining**
- Master inversion, emphasis, articles & prepositions
- Vocabulary: business, interview scenarios + abstract expressions

**Month 6 — Mastering**
- Integrate all grammar, dissect long complex sentences independently
- Self-correct common Chinese-speaker errors
- Vocabulary: remote work scenarios + cultural expressions
- Be interview-ready for remote work settings

## Learning Principles

- **Complex sentences are the focus** — basic grammar is brief; most time is spent analyzing real sentences from tech/work contexts with full word-by-word syntax breakdowns
- **300 words per week** — ~1,200/month, ~7,200 total; every word has 3 common meanings + example sentences
- **Mixed domains** — vocabulary covers internet/tech terms AND daily-life expressions
- **Grammar through analysis** — understand sentence structure by seeing it dissected, not by memorizing rules
- **Review is built in** — each week's file includes a weekend review section

---

## How to Use

1. Start from Month 1 Week 01, or jump to where you need most improvement
2. Read through the grammar section — skim the basics, **focus on the complex sentence breakdowns**
3. Go through all 300 vocabulary words each week — each word has 3 meanings + example sentences
4. Do the sentence-making exercises and compare with the reference answers
5. Complete the weekend review at the end of each week
6. Speaking practice is on you — this plan handles vocabulary, grammar, and sentence output

## Progress Tracking

Each month's Week 04 includes a self-assessment checklist. Use these to track your growth:

- **Week 04** of each month = integration + monthly review + self-assessment + Month N+1 preview
- Mark the self-assessment checkboxes honestly — they help you identify weak spots before the next month
